Les fichiers qui se trouvent dans ce dossier sont des versions en GeoJSON des fichiers de données ouvertes du Ministère de la Culture et des Communications du Québec.

Les fichiers originaux se trouvent sur le site http://www.donnees.gouv.qc.ca

La license d'utilisation de ces données se trouve à cette adresse : http://www.donnees.gouv.qc.ca/?node=/licence

Ces fichiers contiennent une sélection de champs contenus dans les fichiers originaux.
Des champs des fichiers originaux contiennent des valeurs vides.




Contenu des fichiers :


Le fichier " GeoJSON_MCC_IP_CI.json " 
contient les données du fichier original " Donnees_ouvertes_MCC_IP_CI_v1.txt " que l'on retrouve sur le site http://www.donnees.gouv.qc.ca
qui contient des données sur les :
Immeubles patrimoniaux cités par les municipalités et les communautés autochtones


Le fichier " GeoJSON_MCC_IP_CL.json " 
contient les données du fichier original " Donnees_ouvertes_MCC_IP_CL_v3.txt " que l'on retrouve sur le site http://www.donnees.gouv.qc.ca
qui contient des données sur les :
Immeubles patrimoniaux classés par le ministre de la Culture et des Communications

contient les données du fichier original " Donnees_ouvertes_MCC_SP_CI_v1.txt " que l'on retrouve sur le site http://www.donnees.gouv.qc.ca
qui contient des données sur les :
Sites patrimoniaux cités par les municipalités et les communautés autochtones

Sites patrimoniaux classés par le ministre de la Culture et des Communications
